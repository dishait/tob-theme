# tob-theme

tob 的主题仓库

<br />


## 注意事项 👀

需与 [tob-less](https://tob-less.netlify.app/) 结合使用。

<br />
<br />

## 使用 ⏰

git clone 和下载代码后放到项目目录下，再在 `App.vue` 中引入即可。

例如

```html
<!-- App.vue -->
<style lang="less">
@import '@/uni_modules/tob-ui/index.less';

/* 主题请放在 tob-less 后边 */
/* 引入主题 (质感) */
@import '@/theme/texture.less';
</style>
```
